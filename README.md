## Tools yang digunakan:

- React + Vite
  
- Tailwind CSS
  
- Lucide React Icons
  
- Fake Store API
  
- ExchangeRate API

## Cara menjalankan:

1. Clone repository
   
3. Install dependencies: `npm install`
   
4. Jalankan dev server: `npm run dev`
   
5. Buka browser: `http://localhost:5173`

## Fitur

- Browse produk dari API
  
- Filter dan search produk
  
- Shopping cart
  
- Multi-currency support
  
- Payment processing

## API Endpoints

- Products: https://fakestoreapi.com/products
  
- Currency: https://api.exchangerate-api.com/v4/latest/USD
